# Document Similarity Detector

A high-performance text analysis tool to calculate the similarity percentage between two documents. It uses a **Hybrid C++/Python** approach for maximum speed and accuracy.

---

## Key Features
* **Multi-Format Support:** Processes `.pdf`, `.docx`, and `.txt` files.
* **Hybrid Engine:** Uses **Python (Flask)** for the web interface and **C++** for heavy calculations.
* **Smart Algorithms:** Combines **Rabin-Karp (Rolling Hash)** and **LCS (Dynamic Programming)**.
* **Weighted Scoring:** 60% LCS + 40% Rabin-Karp for balanced and symmetric results.

## Tech Stack
* **Languages:** C++11, Python 3
* **Framework:** Flask
* **Libraries:** PyPDF2, python-docx, STL
* **Compiler:** G++
